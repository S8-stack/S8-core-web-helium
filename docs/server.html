Request format

Request format is given below:

  • First 2 bytes: (0-2) Indicate the type of request identification (also, act as an error checking mechanism).
  • Next 8 bytes: (2-10) Give the BkFile address
  • Next 4 bytes: (10-14) Give the BkObject address within the file
  • Next byte: Give the code of the method to be called on the server side object
  • Next bytes: parameters
